LA CASITA

LA CASITA esta totalmente equipada, con heladera, aire acondicionado, TV, termotanque, horno, microondas, vajilla y demás. Tiene capacidad máxima para cuatro personas y parking para un automóvil. Ideal para una familia con chicos.

EL MONO

EL MONO esta totalmente equipado, con heladera, aire acondicionado, TV, termotanque, horno, microondas, vajilla y demás. Tiene capacidad máxima para 2 personas. No cuenta con beneficio de parking, pero si hay disponibilidad se puede estacionar en cochera. Ideal para una pareja.
